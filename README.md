# Browserpad (fork)

This is a fork of the [Browserpad](https://github.com/Browserpad/browserpad) project — an open-source, browser-based plain text editor built entirely with HTML, CSS and JavaScript.

## Fork additions

- Line ending options: **LF** (Unix), **CRLF** (Windows), **DOS** — select before saving
- Character encoding options: **UTF-8** and **ASCII** (non-ASCII chars replaced with `?`)
- Saved files always end with a trailing newline (no "no eol" warnings)
- No localStorage persistence — text is cleared on refresh or close
